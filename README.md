# Rahmatullah Restaurant - Landing Page

A modern, responsive landing page for Rahmatullah Restaurant, built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Adapts to different screen sizes (desktop, tablet, mobile)
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and interactive components
- **Navigation**: Sticky header with smooth scrolling to sections
- **Footer**: Comprehensive footer with contact information and social links

## Tech Stack

- **HTML5**: Semantic markup for structure
- **CSS3**: Styling with Flexbox and Grid
- **JavaScript**: Interactivity and animations

## Project Structure

```
rahamatullah-restaurant-landing-page/
├── index.html              # Main landing page
├── css/
│   └── style.css           # Styles and animations
├── js/
```

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor or IDE (VS Code, Sublime Text, etc.)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open the project folder:
   ```bash
   cd rahmatullah-restaurant-landing-page
   ```

3. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```

## Usage

### Navigation

- Click on the navigation links in the header to smoothly scroll to the corresponding sections:
  - Home
  - About
  - Menu
  - Gallery
  - Contact

### Interactive Elements

- **Hover Effects**: Hover over menu items, gallery images, and buttons to see interactive effects
- **Smooth Scrolling**: All navigation links use smooth scrolling for a better user experience
- **Mobile Menu**: Click the hamburger icon on mobile devices to toggle the navigation menu

## Customization

### Changing Content

To update the content of the website:

1. Open `index.html` in your text editor
2. Edit the text content in the desired sections
3. Save the file and refresh your browser

### Updating Images

To replace the images:

1. Add your images to the `assets/` folder
2. Rename them to match the existing filenames (e.g., `hero-bg.jpg`, `menu.jpg`)
3. Or update the `src` attributes in `index.html` to point to your new image files

### Modifying Styles

To change the design and styling:

1. Open `css/style.css`
2. Edit the CSS rules to customize colors, fonts, spacing, etc.
3. The styles are organized by section for easy modification

## Development

### Adding New Sections

To add a new section to the website:

1. Create a new section in `index.html` with a unique ID
2. Add navigation link in the header pointing to the new section
3. Add corresponding styles in `css/style.css`
4. Add any necessary JavaScript functionality in `js/script.js`

### Adding New Features

To add new features:

1. Create a new HTML section in `index.html`
2. Add CSS styles in `css/style.css`
3. Implement functionality in `js/script.js`
4. Ensure responsive design for different screen sizes
